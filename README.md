# 🍎 Fruit Freshness Classifier (CNN vs SVM vs KNN)

An interactive web application built with **Streamlit** that evaluates and compares three different machine learning models (**Convolutional Neural Network**, **Support Vector Machine**, and **K-Nearest Neighbors**) trained to classify fruit images as **Fresh** or **Rotten**.

## 🛠️ Features
* Upload fruit images dynamically.
* Real-time inference across 3 different models simultaneously.
* Clean visual comparison of classifications and confidence scores.

## 💻 Tech Stack
* **Deep Learning / ML:** TensorFlow, Keras, Scikit-Learn
* **Deployment:** Streamlit Community Cloud
* **Libraries:** NumPy, Pillow (PIL)
